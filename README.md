### Current Project

Building an autonomous drone with GPS-denied return-to-home — inspired by cruise missile guidance (DSMAC/TERCOM), running on a 5-inch racing drone:

- Jetson Orin Nano as companion computer running TensorRT inference in flight
- Fix-and-fly checkpoint navigation: hover, match position against satellite tiles, fly blind on IMU, repeat
- Custom aerodynamic shell designed in Blender with split-fin attachment system
- Full system — from soldering the hardware to writing the AI pipeline
